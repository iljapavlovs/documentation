Accepts a string and returns a lowercase string.

**Parameters**

- `input`: a string.

**Outputs**

Lowercase string.

**Example**

```jpv2
my_lowercase_task [type="lowercase" input="Hello World!"]
```

Given the input `Hello World!`, the task will return `hello world!`.
