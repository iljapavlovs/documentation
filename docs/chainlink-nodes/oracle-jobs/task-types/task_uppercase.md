Accepts a string and returns an uppercase string.

**Parameters**

- `input`: a string.

**Outputs**

Uppercase string.

**Example**

```jpv2
my_uppercase_task [type="uppercase" input="Hello World!"]
```

Given the input `Hello World!`, the task will return `HELLO WORLD!`.
